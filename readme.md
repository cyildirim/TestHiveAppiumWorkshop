# Testhive Meetup |  Mobile Test Automation Workshop

## Requirements

- Appium Desktop (https://github.com/appium/appium-desktop/releases/tag/v1.0.0-beta.5)
- Command line Appium
``` 
$ npm install -g appium
$ appium
```
- Android Studio (To use AVD Devices)
- Xcode 8
